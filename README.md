# QA-FRAMEWORK

This is QA framework for all web based and API test projects. The entire test projects such as hannes-automation, services-automation are childern of this project.
 
   - Java maven project
   - Webdriver / selenium 2 for browser testing and uses Junit for asserstion
   - Reads test data from csv files
   - Implements page object model for code reusability 
   - Spring libraries for API testing

## Precondition

Please insure you have java and maven installed in your machine.

## Run installation

On the QA-FRAMEWORK project run the command below to install the jar file that will be created.

  <pre>
  mvn clean install
  </pre>

## After Installation 
Insure that the maven build was sucess.
The jar file created will be the dependency for: 
hannes-automation and services-automation projects
